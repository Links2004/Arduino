7 This is a simple implementation of multicast DNS query support for an Arduino
8 running on ESP8266 chip. Only support for resolving address queries is currently
9 implemented.
10 
11 Requirements:
12 - ESP8266WiFi library
13 
14 Usage:
15 - Include the ESP8266 Multicast DNS library in the sketch.
16 - Call the begin method in the sketch's setup and provide a domain name (without
17  the '.local' suffix, i.e. just provide 'foo' to resolve 'foo.local'), and the
18  Adafruit CC3000 class instance. Optionally provide a time to live (in seconds)
19  for the DNS record--the default is 1 hour.
20 - Call the update method in each iteration of the sketch's loop function.
